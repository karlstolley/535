Create HTML Automatically

To have your Ruby script generate HTML is not as hard as you would think. You need to include the HTML tags in the Ruby code and concatenate them (+) to the data you were already creating.

To add HTML to the output of a Ruby script, you would:

  • Add lines 21, 22, 29 and 30
  • Add " <td>" +  between puts and each variable on lines 23-28
  • Add  + "</td>" after each variable on lines 23-28
#Version 1: Simple Script to HTML

print "What is the course number of your class? "
number = gets.chomp

print "What is the course title of your class? "
title = gets.chomp

print "Who teaches this class? "
instructor = gets.chomp

print "What is day of the week does class meet? "
day = gets.chomp

print "What time does class start? "
start_time = gets.chomp

print "What time does class end? "
end_time = gets.chomp

puts "<table>"
puts "  <tr>"
puts "    <td>" + number + "</td>"
puts "    <td>" + title + "</td>"
puts "    <td>" + instructor + "</td>"
puts "    <td>" + day + "</td>"
puts "    <td>" + start_time + "</td>"
puts "    <td>" + end_time + "</td>"
puts "  </tr>"
puts "</table>"

Sound too easy? It is. All you needed to do was write the HTML tags in the Ruby script so when it created the results, they would have HTML in them. Nice, huh?

You may have also noticed the changes to each question line (lines 3, 6, 9, 12, 15 and 18). They now use print instead of puts. Change your code to see what this will do.

Save your new Ruby script as create_html1.rb, then add it to your GitHub repository.

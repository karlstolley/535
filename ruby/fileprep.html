Practice Prep

This tutorial will show you how to write a Ruby script, specifically one that will convert a CSV file into an HTML file that you can then include in your mini-web site.

That's great, as long as you have the CSV file!

  1. Download the Ruby script
  2. Open your command line (Terminal)
  3. Type 
    ruby ~/Downloads/create_csv.rb

    (Replace ~/Downloads/ with another path if you downloaded the script to another directory.)

  4. The Ruby Script will ask you:
    1. What do you want to name your file?
      • Do not give it a file extension, the script will set the right one (.csv)
    2. How many classes do you have this semester?
    3. For each class:
      1. What is the Course Number?
      2. What is the Course Title?
      3. Who is the instructor?
      4. What days is it held?
      5. What time does it start?
      6. What time does it end?

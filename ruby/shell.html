Coming Out of Our Shell

Learning Goal

Identify and Use the Interactive Ruby Shell.

With Ruby installed, you get a cool console that lets you try out Ruby code directly on a command line. It is called IRB or the Interactive Ruby Shell.

To start IRB, type irb on the command line.

[~/Projects/ruby/scripts]$irb
irb(main):001:0> def hello name
irb(main):002:1>   puts "Hello " + name + "!"
irb(main):003:1> end
=> nil

For the next few sections, we will use IRB to learn about the basics of Ruby.

Shell
A program allows you to use the operating system's capabilities. Ruby is unlike most programming languages because it has its own shell.

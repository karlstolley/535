Variables

This is usually where people get hooked on Sass: the ability assign variables to items.

Using Variables for Colors

My favorite place to use Sass variables is when declaring colors in Hex or RGB mode: it's difficult to remember what #4C1BE0 means when you look through your CSS rules, and if you haven't been good about using comments, then you have to look it up manually.

Variable Hex Colors

Here is an example of using variables for color in Sass. This is what the normal CSS that is generated by Sass would look like:

body {
  background: #89a194;
  padding: 20px 10px 10px 10px;
  color: #322938;
}
body a {
  color: #a14016;
  font-weight: 900;
}

And here's how the Sass language makes that much more readable:

$green: #89A194;
$grey: #322938;
$orange: #A14016;

body {
  background: $green;
  padding: 20px 10px 10px 10px;
  color: $grey;
}
body a {
  color: $orange;
  font-weight: 900;
}

By assigning the Hex code to a variable, you can keep track of the colors you're using in your CSS rules without having to load the page into a browser.

Variable RGB and RGBA Colors

The same concept applies applies to RGB colors. If you write the color green as:

rgb(27,224,63);

You can assign a variable similar to above for your RGB value:

$green: rgb(27,224,63);

body {
  background: $green;
}

For an RGBA color, you can also assign a variable for the alpha transparency.

$green: rgb(27,224,63);
$alpha: .5;

body {
    background: rgba($green, $alpha);
}

You can even combine the two to make your code even more succinct:

$green: rgb(27,224,63);
$alpha: .5;
$greenAlpha: rgba($green, $alpha);

body {
    background: $greenAlpha;
}

Variables for Fonts

Sass variables aren't constrained to test strings, they can also be used arithmatically, which is a big help when you're defining font sizes for your site. Here is some standard CSS for font sizes:

h1 {
  font-size: 20px;
}
h2 {
  font-size: 18px;
}
h3 {
  font-size: 16px;
}
p {
  font-size: 14px;
}

But by assigning a base font-size in pixels, points, or ems, you can scale it up or down within Sass by using simple arithmatic declarations.

$base-font: 14px;

h1 {
  font-size: $base-font + 6;
}
h2 {
  font-size: $base-font * .95;
}
h3 {
  font-size: $base-font - 4pt;
}
p {
  font-size: $base-font / 1.5;
}

Assigning one base font as a Sass variable allows you to easily make changes to your code later on.

Other Variables

These are just two brief examples of what you can do with Sass variables, but there are two that you can make use of right away. If you want to see more examples, check out the Variables section in the Sass Reference guide.

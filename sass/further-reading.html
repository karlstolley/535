Further Reading

Want to try out Sass for yourself? Follow these steps:

  1. Install Ruby on your computer. Ruby has several different versions available, but if you only need it to use Sass, then just select the most recent stable version.
    • Have a Mac running OSX? Good news: you already have it. Skip to the next step.
    • Have Windows? Use the RailsInstaller.
    • Have Linux? You probably already know what you're doing, but here's some information.
  2. Install the Sass code package (called a "gem"). 
    gem install sass
  3. Technically, you could start using Sass after this step, but since you've already taken the time to install/configure Ruby, you should also install the Compass gem. Compass is a way to automate the conversion of .scss files to .css files. It's definitely worth the extra step. 
    gem install compass
  4. Create a test Compass project
    1. Open the command line tool and navigate to the location where you want your new project created.
    2. Type compass create test-project1.
    3. Navigate to the test-project1 directory that was generated by the Compass gem.
    4. From the same location as above, run the Compass command that will watch for changes to your .scss scripts and automatically convert them into.css files: 
      compass watch test-project1
      This will tell the Compass gem to watch for any changes to the .scss files and convert them to .css files.
    5. Open the test-project1/sass/screen.scss file in your favorite editing program.
    6. Start writing your Sass rules and watch the files in the stylesheets directly get updated each time you save the file.
  5. If you already have a project in place, you can either change your configuration to match the standard Compass/Sass structure, or use the Compass Project tool to change the standard configuration settings.

Using Classes and IDs as Selectors

Think you're limited to using only HTML elements as selectors? Think again.

You can also specify a CSS style using classes and identifiers (IDs). Classes and IDs can be added to HTML elements to indicate specific elements. For example, if you attached a CSS style to a generic <p> tag, all <p> tags would be affected:

Introduction

CSS is where you can turn your plain black-text-on-a-white-background HTML pages into the picture that you have in your head. If HTML is the cake, then CSS is the frosting.

By the End of This Section You Should Be Able To...

The Beginning:

CSS stands for "Cascading Style Sheets." It's the language you use to tell a machine how to display your data. For our purposes, we'll use CSS to tell a web browser how to display our (HTML document ?name of project).

Introduction

CSS is where you can turn your plain black-text-on-a-white-background HTML pages into the picture that you have in your head. If HTML is the cake, then CSS is the frosting.

Learning Objectives

  • Describe what a CSS selector does
  • Describe what a CSS declaration does
  • Use CSS rules to position HTML elements on a page
  • Apply CSS rules to classes and IDs
  • Load CSS from an external stylesheet

Definition

CSS stands for "Cascading Style Sheets." It's the language you use to tell a machine how to display your data. For our purposes, we'll use CSS to tell a web browser how to display our (HTML document ?name of project).

Browsers

Each browser has its own basic style rules it will use. These rules are meant to be overwritten to the web pages' individual rules, but they exist as a sort of failsafe in case a document appears with no rules applied to it at all.

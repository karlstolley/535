Setting up Git and GitHub

Setting up Git and Github is a bit daunting, but you can do it.

First, the Easy Part

Getting started on GitHub is easy. All you need to do is register for a free GitHub account.

Git Setup: Trust the Professionals

GitHub maintains excellent, operating-system specific information for installing and configuring Git. You can find it here; stop following the instructions when you get to the "Password caching" section.

Note GitHub will offer Mac and Windows users the option to download a graphical user interface (GUI) for Git. Do not be tempted. Follow the instructions for installing the regular, command-line version of Git.

SSH Keys

To communicate with GitHub, you'll use SSH keys. Don't worry for now what those are. Just folow these instructions to locate or generate your keys, and add them to your GitHub account.

What Have I Learned?

If you have completed the JavaScript chapter, you should now be familiar with the following topics:

Using Your REPL

A JavaScript console is a built in REPL environment in web browsers that let you evaluate your scripts in real time.

Return to the REPL section.

Basic Programming Concepts

Values, functions, expressions, and variables are the basic building blocks we can use to build programs.

Return to the Programming Concepts section.

Objects

Objects allow us to create key-value pair data structures along with functions that operate on that data.

Return to the Objects section.

The DOM

Learning objects let us learn about the Document Object Model and how we can use JavaScript to manipulate it on our web pages.

Return to the DOM section.

Script Tags and Script Files

Putting our JavaScript programs in separate files and including them on our web pages with script tags allows us to maintain separation of concerns as well as allow for reuse of our scripts.

Return to the Script Tags and Script Files section.

jQuery

jQuery is a JavaScript library that provides us with functions that make it easy to write scripts to manipulate the DOM.

Return to the jQuery section.

Further Reading

The following is a list of good resouces to keep learning more about JavaScript.

  • Eloquent JavaScript

    This book gets into more detail about thinking about programming in general as well as teaching you some cool JavaScript techniques.

  • DOM Scripting

    This is a great resource for really exploring DOM scripting techniques and best practices.

  • JavaScript: The Good Parts

    As you work with JavaScript more, you'll notice that it has some peculiarities. This book is the best resource for helping navigate the tricky parts and writing good, clean code in JavaScript.

  • JavaScript on MDN

    Mozilla Developer Network has several articles and tutorials about different cool things you can do with JavaScript.

  • jQuery Docs

    There's way more cool stuff you can do with jQuery. Browse the documentation to get a taste.

  • JavaScript on DocHub

    DocHub is great for when you know what built-in JavaScript function you want to use, but you don't quite remember what the correct syntax for calling it is. It lets you search by class or function name, and gives you a summary and examples for all of JavaScript's built-in functions.

<table>

why tables Tables are a series of rows and columns that create cells in which data lives.

Tables are built using the <table> and </table> tags. Anything within these tags are part of the table. Tables are built row by row using the <row> and </row> tags. The <table><table> tags indicate the top row heading, which, by default, is bold (can be changed with CSS). The <table> <table> tags contain the table cell data. Our example has a table.

Structure

Body

The body of the webpage houses all of the content that is intended to be displayed to the public. The browser renders everything enclosed within the <body> and </body> tags so that it is viewable. All tags that we learn from this point forward are housed within the body.

Tags Used Within the Body

Div

The <div> </div>is a generic container that is used when none of the other tags appropriately define the information contained. They are used most often with CSS.

Section

The <section> </section>tags are used to group like content together. Sections could be nested or stand alone. Usually, sections contain headers and footers.

Article

The <article> </article> tags is used for grouped information within sections. These are usually not nested unless they are nested within sections of other sections

See this section for more about nesting.

Navigation

The <nav> </nav> tags group navigation links together. These links are usually presented as an unordered list. We will discuss lists in more detail in the next section, but for now it is important to not that all navigation should be distinguished from other parts of the page with the <nav> </nav>.

To see more about links go to Links and Images
.

Footer

The footer comes at the end of a section or the page with extra information.

Paragraph

The <p> </p>tags are probably the most frequently used tags you will see and use in a document, simply because it defines the portion of text that should be displayed as a paragraph. It is possible for a very simple page to be composed simply of <p> </p> elements, however, doing so is very limiting and would not tell you much about the information enclosed. Also, when we start to style the page using CSS, other structural tags, such as <section> and <article> help better define parts of the webpage for clearer and cleaner styling.

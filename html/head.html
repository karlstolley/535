Head

As you can see in the project HTML, the tag comes right after the <html> start tag is the <head> start tag. Look a little below, you can find the <head> end tag. The section enclosed in these tags is called the HEAD of the document and contains important information for the web designer and the browser to know but does not necessarily get displayed for the public. Here, you will include META information (see below) as well as links to stylesheets and scripts (though this could be placed below the body as well). There is no particular order the elements should be presented, if you decided to use any of them, however, the only information that is absolutely necessary to include in the HEAD is the title.

Tags and Elements for the Head

Title

The title tags (<title> and </title>) let the browser know what the name or title of document is. Depending on you browser, the title is typically displayed at the in the title bar of browser window, or if you are using browser tabs, in the tab. It helps orient the user. It also appears as the name of the document in the user's favorites if the user chooses to bookmark the page.

Comments

So what if there is information you want to include for yourself or another developer, and not for the browser? That where comments come in. Comments are extremely useful to the content creator because it helps with reminders about certain code especially when a significant a amount of time lapses in between uses. In our sample project above, we've used comments to let us know how we are defining certain types of meta information. We would also include it to help distinguish types of styles and scripts from one another. To create a comment in html, simple add a <, an exclamation point ! and two dashes -- to form a <!--. Place your comment, then enclose it with two dashes -- and a > . Like all elements in html,comments must be enclosed; in this case, with a start <!-- and ending -->.

Comments are not confined to the head of an html document; they can also be used in the body.

HTML5

Before we continue on to learn about HTML5, we're making the following assumptions.

Assumptions

  • You've heard of the internet
  • You've used the internet
  • You know that websites are not created out of thin air, or worse, Microsoft Word documents (but it's okay if you didn't know)
  • You've read through the Markdown section with reasonable understanding are ready to move on to the next level

Assuming that you know the above (and even if you don't), you by the end of the HTML5 tutorial, you should be able to do the following.

Learning Objectives

  • Know how to create a basic web page
  • Know what an HTML tag, attribute, and element
  • Know what each of the following “tags" do and generally how to use them
    • !DOCTYPE html
    • html
    • head
    • title
    • meta
    • link
    • body
    • header
    • article
    • section
    • footer
    • a
    • img
    • table
    • p
    • nav
    • ol
    • ul
    • li
    • row, etc.
  • Be familiar with each of the following “tags” for future reference
    • aside
    • audio

    • output

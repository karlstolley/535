To Get The Most Out of This Tutorial

  • You are familiar with text editors and browsers (if not go here)
  • You have reviewed the Markdown tutorial with reasonable understanding are ready to move on to the next level

Learning Objectives

  • You will learn how to create a basic web page
  • You will be familiar with HTML5 tags, attributes, and elements
  • You will know what each of the following “tags" do and how to use them
    • !DOCTYPE html
    • html
    • head
    • title
    • meta
    • link
    • body
    • header
    • article
    • section
    • footer
    • a
    • img
    • table
    • p
    • nav
    • ol
    • ul
    • li
    • row, etc.
  • You will be familiar with each of the following “tags” for future reference
    • aside
    • audio
    • details
    • output

We have a lot to learn, so let's get started!

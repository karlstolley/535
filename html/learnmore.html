Behind the Scenes: Source Code

Another way to learn more about HTML is to see it in action on the various websites that you visit. Look behind the scenes at the HTML source code to see how other webdesigners and webdevelopers put their websites together. Keep in mind that only the information that the browser renders will displayed, that is, only the information not processed by the server. Also, sometimes the source code may seem a little complicated with other languages and scripts being added to the mix, so stick with the simpler sites. Like this one!

To view the source code on this webpage,

  • if you are using Firefox,

  • Right-click within the page and from the menu select View Page Source

  • if you are using Chrome,
    1. Click Tools
    2. Click View Source

    Or, simply right-click within the page and from the menu select View Page Source

  • if you are using Safari,...
    1. From the Preference menu, click Advanced
    2. Check Show Develop in Menu Bar
    3. In the Develop menu, choose Show Page Source
  • if you are using Internet Explorer,...
    1. Click View(IE 9) or Page (IE 8)
    2. From the drop-down menu, click Source

Links and Images

HyperLinks

Now, let's learn about what makes a webpage truly a webpage, links. If not for links, a webpage would be a static document isolated from the rest of the internet nearly impossible to reach and just as impossible to leave. Links are what makes the world wide web, the world wide web. Now that we've stressed the importance of learning such a critical component of webpages, we're going to learn about three types of links: in page links or bookmarks, local page links, and external links. But before we do, let's take a closer look at the hyperlink tag and its individual components.

Links are formed with the start tag <a> and end tag </a>. The a in this tag means anchor, and the <a> can refer to either the origin anchor, where the link starts, or the target anchor, where the link ends, on text within the same webpage. The origin anchor, also known as the source anchor, is where the link begins and is identified by the simple presence of the opening <a> tag. The target anchor is identified within the opening tag with an attribute [aside: for a refresher on attributes, visit the Tags, Attributes, and Elements section.] To identify the target anchor, we need to use the href attribute. The value of the attribute will be the destination. Depending on the type of link, the attribute is handled in different ways, as we will soon see below.

In addition, a title attribute could be added to the opening tag to let the user know where the link will take them; the title may appear if you hover the cursor over the link, if enabled, or for accessibility.[see this webpage for accessibility]

In short, the element structure for links is <a href="LINK DESTINATION" title="DESTINATION TITLE"> WHAT VIEWER SEES AS A HYPERLINK</a>. [aside: don't confuse the <a> tag with the <link> tag; <a> tags only appear in the body, and <link> tags only appear in the head of the document]

External Pages

Links to external pages are the simplest of the link types. All you have to do to insert a link to an external page is use the <a> element structure listed above and insert the web address (including http//:) as the destination value. For example, in the project above, we have a link to the IIT home page. The user will see only the IIT highlighted as a hyperlink.

Internal Pages

Internal page links are those found within the same site, though not within the same webpage. For example, the various sections of this tutorial are housed on different pages within the same website- DigitalContentCreatorsResource.com. You have to know where the page you would like to link to lives within the website, in order to link to it using an internal link.

Bookmarks

We refer to bookmarks as links that take you to anchors on the same page. You must define the anchor. Bookmarks are links found in a webpage that take you to other parts of the same page.

Images
